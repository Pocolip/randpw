# randpw
Random password generator made in go

## Build

`go get github.com/pocolip/randpw`

## Usage

`./randpw [password_length]`

## Examples

./randpw 10

oscar lima papa DELTA WHISKEY 7 romeo JULIET VICTOR zulu
